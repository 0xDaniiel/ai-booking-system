# AI-Powered Appointment Booking System

> Full-stack AI-powered appointment booking system featuring an AI scheduler chatbot, admin dashboard, automated reminders, availability calendar, client profiles, and Stripe payment integration.

## Features

- AI Chatbot Scheduler for appointments
- Admin Dashboard to manage bookings & clients
- Email/SMS automated reminders
- Availability Calendar
- Client Profile History
- Stripe Payments

## Tech Stack

- Frontend: React, TypeScript, Tailwind CSS
- Backend & DB: Supabase
- AI: Lovable AI
- Payments: Stripe
